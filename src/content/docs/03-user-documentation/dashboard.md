## Key Features

The Dashboard serves as the central navigation hub for monitoring service performance. It displays comprehensive KPIs for your selected service with the following capabilities:

**Selection Controls**
- Service selection via dropdown menu (top right)
- Duration options: All, 3mo, 6mo, 1y, 2y (top right button set)
- KPI categories: All, Bibliographic, Usage, Technical, Satisfaction (top left button set)

**Performance Indicators**
- KPI cards showing last submission values and percentage change from duration start
- Trend curves for each indicator
- MAPE scores with traffic light system highlighting unstable developments

**Visualizations**
- Trend chart displaying all indicators across the selected duration
- Customizable traces with toggle, y2-axis assignment, and accumulation options

**Actions**
- **Settings**: Save dashboard preferences for service, duration, and KPI category
- **Create Report**: Quick access to PDF export
- **Export Menu**: Download measurements as PDF report or CSV file
- **Announcements**: System and Reviewer notifications displayed prominently
