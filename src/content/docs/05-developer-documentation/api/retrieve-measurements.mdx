  After posting, how do I verify the same KPI values were stored correctly?

</PersonaSays>

Use `GET /denbi/api/v1/measurements` and filter by service, indicator, and date range.

Useful optional filters are `indicators`, `start`, `stop`, `page`, and `pageSize`.

```python
service_abbr = "my-service"
indicator_abbr = "active_users"

payload = api_get(
  "/denbi/api/v1/measurements",
  params={
    "service": service_abbr,
    "indicators": indicator_abbr,
    "start": "2026-03-01T00:00:00",
    "stop": "2026-03-31T23:59:59",
    "page": 0,
    "pageSize": 1000,
  },
)

measurements = unwrap_result(payload)
print(f"Fetched {len(measurements)} rows")
print(measurements[:3])
```
